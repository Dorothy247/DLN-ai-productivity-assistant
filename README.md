# DLN AI – AI-Powered Workplace Productivity Assistant

 **Empowering learners, facilitators, and training organizations through intelligent AI productivity.**

# About the Project

DLN AI is an AI-powered Workplace Productivity Assistant developed during the **AI Skills Acceleration Programme (ASA 10)**.

The project demonstrates how Artificial Intelligence can improve workplace productivity by automating repetitive tasks such as writing emails, summarizing meetings, planning schedules, conducting research, and helping learners quickly access important information.

Rather than replacing existing learning platforms like Microsoft Teams, DLN AI acts as an intelligent assistant that helps users navigate their daily work more efficiently.


# My Inspiration

This project was inspired by a real challenge I experienced during my participation in the AI Skills Acceleration Programme.

Although Microsoft Teams was used to share assignments and learning materials, many learners including myself often became confused about:

- Where to submit assignments
- Where to find workshop slides
- Where announcements were posted
- Which resources belonged to each module
- What deadlines were approaching

Learners frequently asked one another the same questions, which slowed productivity and created unnecessary frustration.

Instead of accepting this as normal, I decided to design an AI assistant that could guide users, automate repetitive tasks, and make learning more organized and productive.

That idea became **DLN AI learning NAVIGATION**.


# Problem Statement

Professionals and learners spend a significant amount of time on repetitive administrative work.

Examples include:

- Writing professional emails
- Summarizing meetings
- Planning daily work
- Conducting research
- Searching for assignments
- Finding learning resources
- Keeping track of deadlines
- Responding to repetitive questions

These repetitive activities reduce productivity and take valuable time away from meaningful work.


#  Solution

DLN AI provides a single intelligent workspace where AI assists users with everyday productivity tasks.

The application combines multiple AI-powered tools into one modern interface to help users work faster, stay organized, and access information easily.
